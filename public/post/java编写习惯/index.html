写好变量

大小写:变量小写，方法小写，类名大写 长度:单子个字字母的变量只能在3/5内解决 含义:少用temp,it,do等没有意义的词；界面组件用匈牙利命名法（btnSayHello）

使用中间变量

经常使用的一个写起来比较长的变量，写成中间变量。

写好常量

不要从天上掉下了一个数: 1.常量用final,static 2.字符串常量如BorderLayout.CENTER 3.从配置中获取 4.使用java.util.Properties类的load()
